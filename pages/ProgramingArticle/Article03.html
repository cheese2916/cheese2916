マウスの取得～ボタン設計

今まで使ってきたUnityにならって、GameObjectというクラスを設計してみようと思います。
今回は簡単に、UpdateとRenderのみを関数として実装します。
雑にはなりますが、小規模かつ一度は完成させたいという思いからこのような設計になりました。

さて、下記にコードを記します。
GameObject.hpp
#pragma once

#include "IUpdatable.hpp"
#include "IRenderable.hpp"

class GameObject : public IUpdatable, public IRenderable
{
public:
	virtual ~GameObject() = default;
};

以上がGameObjectの設計図になります。
Updateの関数及び、Renderの関数はインターフェースとして実装してあります。
それを継承する形で制作してあります。

次にボタンを作っていくのですが、ボタンを押す際にマウスの入力判定が必要なので
先にマウスの入力検知機を設計しておきます。

Mouse.hpp
#pragma once

#include "DxLib.h"
#include "Vector2.hpp"
#include "GameObject.hpp"

class Mouse : public GameObject
{
public:
	void Initialize();

	void Update() override { Control(); }
	void Render() const override;

	const Vector2& GetPosition()const;
	const bool GetLeftInputState()const;
	const bool GetRightInputState()const;
	const bool GetMiddleInputState()const;
protected:
private:
	void Control();

	Vector2 m_position{ 0,0 };
	int m_mouseInput = 0;
	int m_logType = 0;

	bool m_isLeftPushed = false;
	bool m_isRightPushed = false;
	bool m_isMiddlePushed = false;
	//---Debug---//

	int m_debugTextColor = 0;
	int m_fontHandle = 0;
};

簡単に左クリック、右クリック、中クリックの入力検知を用意しました。
また、入力検知位置及び入力タイプの判定には、GetMouseInputLog2()を利用しています。
